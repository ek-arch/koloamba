## Scoring model as of end of session

| Platform | Auto-score cap | Formula |
|---|---|---|
| X | 30 | `engagement × (1 + min(TS/50, 1) × 2)` |
| Reddit | 10 | `(ups + comments × 1.5) / 50` |
| Telegram | 1 auto / 3 with moderator | flat 1.0 + moderator quick-pick {0.5, 1, 2, 3} |

Leaderboard caps `total_points` at 100. **Tier multipliers retuned later in the session**: now 1.0 / 1.3 / 1.7 (bronze / silver / gold), bands 0–1,999 / 2k–49,999 / 50k+ KOLO.

Two independent multipliers apply:

- **Credibility boost** (from TwitterScore, 1.0× – 3.0×): multiplies X post engagement only
- **Tier multiplier** (from KOLO balance, 1.0 / 1.3 / 1.7): multiplies the final weighted score across all platforms at leaderboard rollup

They stack at the end but never touch each other's inputs — surfaced explicitly across the UI in the second half of the session.
